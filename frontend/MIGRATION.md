## Troubleshooting

If you encounter issues:

1. Clear Next.js cache:
   ```bash
   rm -rf .next
   npm run dev
   ```

2. Reinstall dependencies:
   ```bash
   rm -rf node_modules
   npm install
   ```

3. Check for missing `'use client'` directives in components using hooks
